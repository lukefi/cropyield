Preparation¶

Linux system (windows with linux subsystem may work but no support is offered for that). Testing done on Ubuntu 16.04, with Python 2.7 and 3.7 and Anaconda 4.6.14 and Puhti with geoconda module.

Puhti¶

For running the scripts on Puhti (CSC), a useraccount with BU has to be available. The scripts call the geoconda module. The billing project for the array jobs has to be given in config.config (puhtiproject).

Workstation¶

Python libraries used:
  • os
  • sys
  • csv
  • shutil
  • subprocess
  • argparse
  • datetime
  • numpy
  • rasterstats
  • gdal
  • shapely
  • fiona

Some of above mentioned are not default libraries and have to be installed separately. Creating an anaconda environment is suggested. A .yml file is included in the package of which a new environment can be created.

Run

conda env create -f environment.yml

to create the environment. See here for more information on the topic.
